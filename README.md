# Polyhedral Game Framework — Documentation

Source for the Polyhedral Game Framework documentation site:
**<https://monokkel.github.io/polyhedral-docs/>**

The framework is a collection of modular Unreal Engine 5 plugins for turn-based,
board-game-like games (deckbuilders, tactics games, CRPGs).

- Built with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/);
  deployed to GitHub Pages on every push to `main`.
