# Community Guides

This section hosts guides written by users of the Polyhedral Game Framework —
walkthroughs, integration recipes, patterns that worked in your project, or
anything else other users would benefit from.

!!! info "No guides yet"
    The framework's documentation is still being built out. Once it stabilizes,
    contributions here will be actively welcomed.

## How to contribute a guide

1. Fork the [docs repository](https://github.com/Monokkel/polyhedral-docs).
2. Add a Markdown file under `docs/community/` (one file per guide) and list it
   in the `nav:` section of `mkdocs.yml`.
3. Put a byline at the top of your guide, e.g. `*By [YourName](https://github.com/yourname)*`.
4. Open a pull request. It will be reviewed for technical accuracy and clarity,
   then merged with your byline intact.

See [CONTRIBUTING](https://github.com/Monokkel/polyhedral-docs/blob/main/CONTRIBUTING.md)
for formatting conventions and the content license (CC BY 4.0).
